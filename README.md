Make sure you have a local llm installed. Default is llama3.2:1b. 


If you want to change to any other model. Open folder in terminal and run
```
cp .env.example .env
```
Then paste in terminal
```
OLLAMA_MODEL=<YOUR_MODEL>

```
INSTALL DEPENDENCIES
```
npm install

```
START THE SERVER
```
npm start
```
CHAT WITH AI(in another terminal)
```
node chat.js
```
